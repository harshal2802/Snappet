What you're looking at

Nodes are colored by type (app, hook, build step, dependency, doc…). Edges show how things connect: solid = containment/usage, dashed = a localStorage write, dotted = a deploy hop. Click any node to see its details and neighbors. Use Trace path to find how two nodes are wired together.
