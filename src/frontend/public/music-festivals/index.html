Festival lineup packs for the Snappet Festival app. Public schedule data, community-maintained, no accounts, no tracking. In the app: Festival → Download from Snappet Lineups — or download a .fpack here and AirDrop it (Import pack file…).

Format: .fpack = "FPAK" magic + 1 version byte + raw-DEFLATE JSON — { festival, days[] → stages[] → sets[] (artist, start, end) }, validated by the same rules the app's installer enforces (06:00 day rollover, no same-stage overlaps, festival-local ISO times). Set times are the published schedule; your clips tag against them on-device — nothing about you ever reaches this site.

Machine-readable: manifest.json — the index the app's browse screen consumes.
